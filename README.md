# Edro Book Store

This is a frontend book store built using React + TypeScript + Vite.

## Getting Started

### Using docker/docker-compose

1 - Clone the repo

2 - CD to the root of the project.

3 - In the terminal, execute the following commands:

  ```bash
  docker compose up
  ```

### Using npm

1 - Clone the repo

2 - CD to the root of the project.

3 - In the terminal, execute the following commands:

  ```bash
  npm install
  npm run dev
  ```
